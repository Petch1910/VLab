# Product Spec

## Product

โปรแกรม Vanguard ภาษาไทยสำหรับคอมและมือถือ มีฐานข้อมูลการ์ด, deck builder, สนามเล่น, replay และ bot

## Target Users

- ผู้เล่น Vanguard ไทยที่อยากค้นการ์ดและจัดเด็ค
- ผู้เล่นที่อยากซ้อมกับบอท
- ผู้พัฒนาที่อยากทดลอง rule/bot/deck

## Core Features

### Card Browser

- ค้นชื่อไทย/รหัส/ซีรีส์/แคลน/เนชั่น/เกรด/text
- แสดงรูปและรายละเอียด
- filter ตาม format

### Deck Builder

- เพิ่ม/ลบการ์ด
- counter Main/Ride/G
- validator
- save/load
- deck code
- export deck image

### Manual Play Table

- zones ครบตาม Vanguard
- manual actions
- undo
- action log
- replay

### Bot Fight

- bot heuristic
- difficulty profiles
- NPC campaign ภายหลัง

### Custom Packs

- import `.csv/.xlsx + images.zip`
- manifest/hash verification

### Online Future

- room fight
- spectator
- sync ผ่าน event log

## Non Goals For MVP

- ranked online
- full automatic effect resolution
- official game clone
- monetization
