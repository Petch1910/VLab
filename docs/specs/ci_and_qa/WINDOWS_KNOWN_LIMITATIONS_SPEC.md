# M27-07 Windows Known Limitations Spec

## Goal

Create a concise local Windows build limitations list so future work does not
confuse foundation features with complete product features.

## Scope

The limitations list must cover:

- product scope
- card data
- PlayTable
- online
- bot / automation
- performance
- release boundary

## Output

`docs/WINDOWS_LOCAL_BUILD_KNOWN_LIMITATIONS.md`

## Verification

Docs-only verification:

- limitation document exists
- it states Windows-only scope
- it states no public release without user instruction
- it states trusted-client online limitation
- it states ability automation limitation
