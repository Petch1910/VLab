# M33-03 Combo Matrix Artifacts Closeout

## Summary

Built matrix artifacts from the M33 combo discovery reports so combo density can
be reviewed by era, clan/nation group, top score, and synergy tag.

This remains an offline advisory analysis layer. It does not mutate
`GameState`, consume RNG, resolve card text at runtime, or wire anything into
bot decisions.

## Matrix Meaning

- `combo_matrix_era_summary.csv`
  - rows: era presets
  - values: card count, group count, nonempty group count, candidate-pair
    count, missing set count
- `combo_matrix_group_candidates.csv`
  - rows: clan/nation groups
  - columns: era presets
  - values: thresholded candidate-pair counts
- `combo_matrix_group_cards.csv`
  - rows: clan/nation groups
  - columns: era presets
  - values: card counts
- `combo_matrix_top_pair_scores.csv`
  - rows: clan/nation groups
  - columns: era presets
  - values: highest stored top-pair score
- `combo_matrix_synergy_tags.csv`
  - rows: preset + group
  - columns: synergy tags
  - values: counts among each report's stored `top_pairs`

Full card x card matrices are intentionally not generated by default because
they would be large and noisy. The useful review flow is:

1. read group/era density matrix
2. inspect high-density groups
3. inspect top pairs in the matching preset report
4. promote selected pairs only after review
