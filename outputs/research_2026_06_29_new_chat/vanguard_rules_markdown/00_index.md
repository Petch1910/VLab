# Cardfight!! Vanguard Rules Markdown Corpus

This folder contains page-by-page Markdown extractions of official Cardfight!! Vanguard Comprehensive Rules PDFs, plus indexes for studying rule evolution by era.

Important: these Markdown files are text extractions. PDF icon glyphs can be missing or rendered as blanks by text extraction, so source links and SHA-256 hashes are included for verification against the original PDFs.

Canonical source registry for the development docs: `../vanguard_development/source_references.md`

Each version file includes source PDF URL, local PDF path, SHA-256, detected version, detected update date, page count, and extracted character count in its metadata block.
