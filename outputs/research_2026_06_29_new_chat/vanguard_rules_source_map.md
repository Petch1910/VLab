## วิธีอ่านที่แนะนำ

ไม่ควรอ่านไฟล์ล่าสุดรวดเดียวตั้งแต่ต้น เพราะ Comprehensive Rules ล่าสุดรวมระบบทุกยุคเข้าด้วยกัน วิธีอ่านที่ง่ายกว่าคือ:

1. อ่าน `1.19` เฉพาะ core rules ก่อน
   - winning/losing
   - zones
   - ride/call
   - attack/guard
   - drive check/damage check
   - trigger
   - Counter-Blast / Soul-Blast / Soul-Charge

2. กลับมาอ่าน `1.19` ส่วนระบบยุค Link Joker
   - Limit Break
   - Lock / Unlock

3. อ่าน `1.31`
   - Legion
   - Seek Mate

4. อ่าน `1.38`
   - Stride
   - Heart
   - Generation Break

5. อ่าน `1.44.1` และ `1.46.4`
   - G Guardian
   - clan keywords ยุค G เช่น Brave, Ritual, Unite, Thunderstrike, Darkness, Wave

6. อ่าน `1.48`
   - Ultimate Stride
   - Zeroth Dragon / Gyze logic

7. อ่าน `2.01` แล้วข้ามไป `2.3`
   - Imaginary Gift
   - Front Trigger
   - V rules

8. อ่าน `4.45` หรือ `4.52/4.55`
   - Ride Deck
   - Persona Ride
   - Over Trigger
   - overDress / XoverDress
   - Order / Set Order
   - Energy
   - Divine Skill
   - D/DZ-specific rules

## หมายเหตุ

- ไฟล์ `Cardfight-Vanguard-Comprehensive-Rules.pdf` เป็น official upload แต่ไม่ใช่ภาคแรกแท้ ๆ; เป็น ver. 1.19 ปี 2014 จึงมี Limit Break/Lock แล้ว
- ถ้าต้องการ “ก่อน Limit Break” จริง ๆ อาจต้องใช้ mirror รุ่นเก่ากว่า เช่น ver. 1.10 ที่เจอตามเว็บภายนอก แต่ความน่าเชื่อถือต่ำกว่า official upload
- สำหรับการเรียนและทำฐานข้อมูล ใช้ official versions ข้างบนก็พอ ครอบคลุมระบบหลักตั้งแต่ OG ถึง DZ
