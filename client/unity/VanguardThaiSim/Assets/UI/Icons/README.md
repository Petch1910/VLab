# UI Icons

This folder stores selected open-source icons for the Vanguard Thai Sim UI.

Current source:

- `Lucide/` contains a small selected subset from `lucide-static@1.21.0`.
- License and package metadata are stored inside `Lucide/`.

Rules:

- Do not copy icons or UI assets from VangPro, Vanguard Area, Dear Days, or
  other commercial/proprietary games.
- Do not import full icon packs into Unity.
- Add every new icon to `docs/UI_ICON_SYSTEM_SPEC.md`.
- If a new source is added, include its license and attribution requirements.
