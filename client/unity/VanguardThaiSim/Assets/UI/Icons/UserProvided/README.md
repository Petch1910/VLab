# User-Provided Icons

This folder is reserved for private icon overrides.

Codex must not place proprietary or extracted commercial assets here.

If you already have icon files that you are allowed to use, place them here and
map them in `icon-pack-manifest.json`.

Expected first-pass names:

- `trigger_critical.png`
- `trigger_draw.png`
- `trigger_front.png`
- `trigger_heal.png`
- `trigger_over.png`
- `trigger_stand_legacy.png`
- `marker_force.png`
- `marker_accel.png`
- `marker_protect.png`
- `marker_quick_shield.png`
- `marker_persona.png`
- `marker_crest.png`
- `marker_energy.png`
- `marker_gauge.png`

Use transparent PNG files at `64x64` or `128x128` for best compatibility.
